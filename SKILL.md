# 吴晓波频道风格写作 Skill

## 账号画像

**吴晓波频道** — 新中产财经内容平台
- 核心定位：帮助新中产理解经济与商业世界
- 受众：30-50岁中产阶层、企业家、财经爱好者
- 调性：**温度 + 故事感 + 历史纵深 + 个人视角**
- 内容主轴：财经知识 + 产业观察 + 企业故事

---

## 核心写作特征

### 1. 标题公式

```
【叙事感型】（最常用）
→ "[公司/现象]，一个[时间跨度]的故事"
→ "当[主体]遇到[挑战]，它选择了[出人意料的路]"
→ "[历史事件/人物]，和[当下现象]惊人地相似"

【情感钩子型】
→ "那些[消失/改变的东西]，我有点舍不得"
→ "[这代人]正在经历一种新的[情绪/处境]"
→ "[平凡的人]，如何在[大时代]里找到自己的位置"

【双关判断型】
→ "[数字]的背后，是[深层情感或社会现象]"
→ "为什么[某事]，比我们想象的更重要"
```

### 2. 文章结构

```
【开场：用一个故事或场景代入】
— 不从数据开场，从一个具体的人/场景/历史片段开场
— 建立情感连接后，再引出宏观话题
— 开场约150-300字，温度重于效率

【历史纵深段：「这件事从哪里来」】
— 追溯事件/现象的历史根源（5年/10年/20年前）
— 用「那一年」「彼时」「回到XXX年」的叙事手法
— 体现作者的历史感与时代感

【当下拆解段：「现在发生了什么」】
— 呈现当下事件/现象的关键细节
— 可以引人物故事（采访/传记/报道中的真实人物）
— 语气克制，不唱好也不唱坏，但有明确立场

【个人判断段：「我认为」】
— 吴晓波频道特色：给出作者/频道的明确个人判断
— 句子短，判断明确
— 可以带温度，但不能鸡汤化

【结尾：留给读者的思考或情绪】
— 不强调行动号召
— 留一句有余韵的话
— 可以引用历史典故或诗句（但要真实克制）
```

### 3. 写作风格原则

**必须做到**：
- 文章里至少有一个「故事」（真实人物/历史事件/亲历场景）
- 句子有节奏感：长短句交替，避免连续长句
- 有明确的「我们/你/这一代人」的群体感
- 财经内容必须对外行也要讲得通（面向非专业读者）

**绝对不能**：
- 全是数据图表，没有故事和温度
- 观点摇摆，没有立场
- 鸡汤化（「相信自己，一切皆有可能」）
- 只讲成功案例，回避失败与代价

## STEP 流程

### STEP 1：确认参数
```
请告诉我：
1. 主题：经济现象 / 企业故事 / 时代观察 / 人物特写？
2. 目标情绪：让读者感到「原来如此」/ 「值得警惕」/ 「令人感慨」？
3. 有没有你想放入的真实故事或人物？
4. 字数：1500 / 2500 / 3500字
```

### STEP 2：找「故事线」
在开始写作前，确定文章的故事入口：
- 从哪个真实人物或场景切入？
- 历史纵深可以追溯到哪里？
- 「这件事和我们有什么关系」的情感连接点是什么？

### STEP 3：信息采集
重点渠道：
- 中国商业史/企业传记/财经著作
- 企业家采访/自传
- 宏观经济数据（国家统计局、中国人民银行报告）
- 国内外财经媒体深度报道

### STEP 4：按骨架写正文
遵循吴晓波频道「故事先行+历史纵深+个人立场」规范

### STEP 5：自检清单
```
□ 开头是否有一个具体的故事或场景，而不是数据？
□ 文章是否有历史纵深（追溯至少5年前的根源）？
□ 是否给出了作者的明确个人立场？
□ 财经内容非专业读者能读懂吗？
□ 结尾是否有余韵，而不是空洞口号？
□ 全文有没有鸡汤化倾向？（有则删除）
```
