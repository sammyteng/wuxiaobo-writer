# 📖 吴晓波频道 Writer Skill

> 让 AI 以吴晓波频道「新中产财经叙事」风格写商业故事的可复用 Skill。

## 简介

本 Skill 基于对吴晓波频道数百篇财经叙事文章的结构化解析，将吴晓波频道「以讲故事口吻拆解商业现象、有历史纵深的个人视角」标准化为可复用 AI 写作指令。

## 文件结构

```
wuxiaobo-writer/

    ├── playbook_history.md         # 骨架二：历史纵深报道
    ├── playbook_person.md          # 骨架三：企业家人物特写
    ├── playbook_era.md             # 骨架四：时代观察与评论
    └── playbook_industry.md        # 骨架五：产业变迁解读
```

## 吴晓波频道核心写作风格

| 维度 | 特征 |
|------|------|
| 开场 | 用故事/场景代入，不以数据开场 |
| 历史感 | 追溯5-20年历史根源，「那一年」「彼时」 |
| 个人视角 | 明确给出作者/频道立场 |
| 受众 | 新中产（财经内容必须对非专业读者通俗） |
| 禁忌 | 鸡汤化、数据堆砌无温度、观点摇摆无立场 |

## 核心写作哲学

- **故事先行**：先讲故事建立情感连接，再引出宏观分析
- **历史纵深**：用历史视角看当下，体现作者的时代感
- **温度与态度**：「理性而克制」地表达情绪，有立场不煽情
- **新中产语言**：专业内容用平实语言讲清楚，面向有一定认知的非专业读者

## 如何使用

```
使用 wuxiaobo-writer，帮我用吴晓波频道风格写一篇关于 [主题] 的财经文章
```

## 适合场景

✅ 重大商业事件的叙事解读
✅ 企业家/时代人物故事
✅ 中国商业史回顾与当下对照
✅ 新中产关注的经济生活话题
