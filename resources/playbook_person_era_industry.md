# 骨架三：企业家人物特写

适用场景：以有温度的方式写一位企业家/创业者的故事，关注其内心世界与时代处境。

---

## 结构模板

```
【一个让读者记住这个人的细节】
不从简历开始。用一个细节建立人物印象：
- 他们的一个习惯或执念
- 一个让外人觉得奇怪但他们觉得理所当然的选择
- 他们身上「和普通人不同」的那一面

【这个时代如何塑造了他/她】
时代背景 × 个人性格的交织：
- 是哪个历史时刻/社会环境造就了他们的价值观？
- 他们的创业，是时代给的机会，还是逆时代而行？

【最重要的一次选择】
找出这个人商业生涯中「命运转折点」：
- 放弃了什么？得到了什么？
- 当时有多少人不理解那个选择？
- 现在来看，对了吗？

【他们的代价与收获】
不只写成功：
- 为了这条路，他们放弃了什么（家庭/健康/理想）？
- 有没有曾经后悔过的瞬间？
- 这些代价值得吗？（留给读者判断）

【他们眼中的下一个问题】
结尾聚焦当下：
- 这个阶段，他们最在意的事是什么？
- 用他们自己的行为（而非语录）来呈现
```

---

## 写作要点

- 必须有细节，不能只讲商业成就
- 必须呈现人物的局限性/代价，不只是赞颂
- 「时代塑造」要具体（某年的政策/事件），不要笼统说「那个年代」

# 骨架四：时代观察与评论

适用场景：对一个重大经济或社会现象做有温度、有立场的时代性评论。

---

## 结构模板

```
【开场：一件让你感慨的事】
用作者/频道的「第一视角」切入：
「我最近[观察/听到/看到]了一件事，觉得值得认真说说。」

【这件事说明了什么时代特征】
从具体现象出发，提炼时代信号：
- 这件事为什么现在发生（而不是10年前/10年后）？
- 它预示着什么更大的变化？

【支撑判断的数据与故事】
数据 + 故事双线并行：
- 宏观数据说明趋势的规模
- 具体故事说明趋势对真实人产生的影响

【作者的明确立场】
吴晓波频道必须有立场：
- 「我们认为，这件事对[某类人群]意味着[判断]」
- 不回避争议，但措辞克制

【给读者的「行动或反思」】
结尾不是口号：
- 给读者一个从自身出发可以思考/行动的角度
- 「面对[时代趋势]，[某类人]可能需要重新想想[某事]」
```

# 骨架五：产业变迁解读

适用场景：用历史感和当下数据结合，解读一个产业的长周期演变逻辑。

---

## 结构模板

```
【产业「黄金年代」的样子】
先描述这个产业曾经「蓬勃」时的状态：
- 哪些数字代表巅峰？
- 那时候「入行」意味着什么？

【什么改变了一切】
找出引发产业变迁的关键变量（1-2个）：
- 技术替代 / 需求转移 / 监管冲击 / 竞争格局重组
- 这个变量是何时出现的？有多快？

【变迁中的人】
用具体人物来呈现宏观变化对个体的影响：
- 谁因此受益？谁被淘汰？
- 他们如何应对这场变迁？

【产业的当下地图】
现状呈现：
- 这个产业现在的体量/利润率/主要玩家
- 和鼎盛时期相比，什么变了，什么没变

【产业的下一个坐标】
不是「一定会发生」，而是「如果[条件]，可能出现[方向]」
```
